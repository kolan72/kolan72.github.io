Последняя версия
2.6.0.5
  • Оптимизирована скорость загрузки заставки.
  • На форме заставки отображается ход загрузки приложения.
  • Исправлена ошибка - форма настройки не открывалась из боковой панели словаря.
  • Исправлена ошибка, когда выделенное на странице слово, которое добавляется в форму Быстрой фиксации слов, не попадало в нее, если эта форма уже открыта.
  • Улучшена навигация по ссылкам на странице.
  • Скорректированы некоторые подсказки к кнопкам.
Предыдущие версии

2.6.0.0

  • Добавлена поддержка перетаскивания ссылки из другого браузера.
  • При смене вкладки отображается отслеживаемое слово, если раньше произошел переход на него на этой вкладке.
  • Исправлена ошибка закрытия других вкладок через контекстное меню.
  • Исправлено то, что в режиме Повтора не было возможности перейти на слово из Словарей - теперь слово из Словарей открывается в другой вкладке.
  • Добавлена всплывающая полупрозрачная форма, с кнопками Быстрой фиксации слов и Добавить слово, как в панелях инструментов.
  • Добавлена возможность перетаскивания вкладки, т.е. изменение относительного порядка вкладок.
  • Исправлено то, что при добавлении слова в Отслеживаемые через боковую панель либо через форму Быстрой фиксации слов слово не было видно при последующем клике на соответствующую закладку.
  • Исправлено то, что при переходе на другое слово из поля поиска онлайн-словаря отслеживаемое слово оставалось видимым, если раньше произошел переход на него на этой вкладке.
  • Оптимизировано использование памяти программой.

2.5.0.2

  • Добавлена возможность помечать слова как Отслеживаемые и работа с такими словами.
  • Изменена форма Настройки повтора слов для работы с Отслеживаемыми.
  • В Словарь добавлены:
    • колонка Дата добавления.
    • кнопка "Повторить".
    • выпадающий список "Тип повтора" и возможность удалять из отслеживаемых.
  • Кнопка "Быстрая фиксация слов" продублирована на боковой панели словаря.
  • Исправлены ошибки
    • отображения иконки сайта при переходе между вкладками.
    • отображения ссылок при повторе слов.
    • сохранения всех открытых вкладок в закладке, когда выделено слово вместо даты.
    • перехода по ссылке в Словаре.

2.0.0.0

  • Добавлена возможность быстрой фиксации слов.
  • Добавлены следующие возможности по работе с Закладками:
    • Поиск в Закладках
    • Удаление нескольких слов из папки с помощью меню "Управление словами".
    • Сохранение сразу всех открытых вкладок.
    • Закрепление папки с датой для последующего добавления в нее слов-ссылок в форме "Добавление в Закладки"
    • В контекстное меню папки с датой добавлено подменю "Обновить".
  • В работе со Словарем добавлены:
    • Возможность настроить работу с кнопкой поиска Словаря на боковой панели так, чтобы при клике на ней искомое слово сразу открывалось на сайтах поддерживаемых онлайн-словарей
    • Возможность удаления слова из формы Словаря
    • В Словаре на боковой панели добавлена всплывающая подсказка на сайт онлайн-словаря при наведении мышью на слово.
  • Добавлены следующие возможности по работе с Повтором слов:
    • Повторение слов только выделенной папки
    • Исключение слова из текущего повтора.
  • В работе со вкладками добавлены:
    • Возможность закрыть другие (неактивные) вкладки.
    • Возможность закрытия всех вкладок.
    • Возможность закрытия активной вкладки по двойному клику.
  • В Настройки добавлена вкладка Разное для возможности выбора провайдера иконки сайтов
  • Добавлена возможность закрытия активной вкладки после добавления закладки
  • В Меню Вид > Боковая панель добавлено подменю "Словарь"
  • Исправлены мелкие ошибки дизайна
