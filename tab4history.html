Последняя версия
3.0.0.0
  • Добавлена возможность присваивать тэги папкам назначения и фильтровать по ним.
  • Добавлена возможность группового переименования.
  • Исправлены ошибки, связанные с отменой редактирования карточек команд и папок.
  • При удалении файлов через Shift+Del теперь происходит запись в лог и отображение информации в строке состояния.
  • Исправлена ошибка позиционирования на текущую папку в окне выбора папки.
  • Ужесточено требование проверки при редактировании соответствия папок - не должно быть дублирования коротких имен.
  • Исправлено появление множественных сообщений при повторных ошибочных сообщениях.
  • После команды "Копировать и удалить" для следующих команд файлы этой команды недоступны
  • Исправлена ошибка, появляющаяся после удаления при позиционировании курсором на последнюю запись.
  • При удалении файлов через Shift+Del в случае ошибки выдается сообщение.
  • Появление всплывающей формы с миниатюрой при наведении на иконку файла стало позиционироваться более корректно.
  • Добавлена возможность перетаскивания пути к папке из проводника Windows - в главной форме и карточки папки.
  • Программа адаптирована для средних шрифтов.
Предыдущие версии

2.5.0.0

  • Добавлена файловая команда Копировать и удалить.
  • Улучшена навигация по приложению с помощью быстрых клавиш, в частности, теперь между списками файлов и папок назначения можно перемещаться с помощью клавиши Tab.
  • Цвета выделенных строк в таблицах файлов, папок и команд различаются в зависимости от того, является ли активной соответствующая таблица.
  • Исправлена ошибка отображения некорректной информации в строке состояния при отмене выполняемой файловой команды или наличии ошибок.
  • Отметить флажком выделенные строки в таблицах можно клавишей Enter. Снять отметку - клавишами Shift+Enter.
  • Исправлена ошибка, когда пункты меню управления комадами были активны во время применения файловых команд.
  • В Настройках сохраняется сортировка папок назначения.
  • Исправлена ошибка возможности удаления строк из таблицы файлов(без реального удаления файлов на диске).
  • Исправлена ошибка при выделении или показе миниатюры для удаленного файла.
  • Добавлена возможность удаления отмеченных файлов по комбинации клавиш Shift+Del.
  • Между запусками программы сохраняется ширина файловой панели и размер формы(нормальная или максимальная).

  • При изменении пути на сохраненный в выпадающем списке не надо нажимать Enter.
  • Добавлен скроллинг по сохраненным путям.
  • Исправлена ошибка редактирования целевых папок.
  • При изменении пути на сохраненный в выпадающем списке теперь не надо нажимать Enter.
  • Для файлов, имеющих цифры в названиях, сделана естественная сортировка.
  • Сортировка оптимизирована.

  • Миниатюра появляется также при наведении на флажок.
  • Добавлен справочник целевых папок.

  • Добавлена кнопка Показать содержимое папки.

  • После применения команд показывается лог.

  • Исправлена ошибка отображения миниатюры, когда основная форма не активна<./li>
  • Убрано неуместное всплывание миниатюры
  • Добавлены горячие клавиши для кнопок Обновить и Применить.
  • Фильтр по имени файла стал зависимым от ширины колонки.
