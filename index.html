О программе

TMFExplorer (сокр. от TooManyFilesExplorer) - программа для работы с директориями, в которых слишком много файлов.

Когда количество файлов в директории становится слишком большим, в нее, как правило, невозможно перейти, или программа, которая используется для файловой навигации, не реагирует на действия пользователя, "зависает".

Основное преимущество программы TMFExplorer - возможность гибкой настройки фильтров при выборке файлов из таких директорий, что приводит к умеренному потреблению ресурсов операционной системы и отсутствию "зависаний", а также автоматизация таких операций, как копирование, создание Zip-архивов и удаление файлов.

Системные требования

Операционные системы: Windows XP, Windows 7.

Дополнительно: Microsoft .Net Framework 4.

Описание работы

Основное назначение программы TMFExplorer - работа с директориями, содержащими слишком большое количество файлов. Путь к такой директории нужно задавать, раскрыв левую панель "Исходная директория". На левой же панели расположены вкладки фильтров по файлам при обновлении, обработке и удалении файлов из исходной директории. Ниже представлено подробное описание операций обновления и обработки. Полный функционал программы представлен в разделе "Функциональность"

Обновление исходной директории

При открытии программы файлы из исходной директории не отображаются. Для обновления исходной директории необходимо нажать кнопку "Обновить", предварительно настроив фильтры, если в этом есть необходимость.

Время обновления может оказаться существенным и определяется доступностью ресурса исходной директории, количеством файлов, заданными фильтрами, и пр. Количество попыток обновления задается в Настройках программы. В процессе обновления его можно отменить, либо, через некоторое время после начала обновления, поставить на паузу.

По окончании обновления раскрывающаяся панель "Показать файлы" станет доступной и отобразит найденные файлы и некоторую дополнительную информацию по ним. Директории на левой панели не отображаются.

Обработка исходной директории

Под обработкой понимается последовательность из одной или более операций с файлами исходной директории, выполняемых друг за другом. Операции выбираются в левой панели в группе "Последовательность обработки". В случае выбора более одной операции относительный порядок их выполнения опреляется взаимным расположением - сначала выполняется операция, расположенная выше.

Состояние выполнения операций отображается в раскрывающейся панели "Лог обработки".

Фильтры применяются только к операции копирования в целевую директорию.

Операция "Создать Zip-архив файлов целевой директории" применяется ко всем файлам, находящимся на момент начала данной операции в целевой директории, безотносительно к другим возможным операциям текущей обработки. Операции копирования в целевую дирректорию и удаление в исходной по целевой допускают задание нескольких попыток. При выполнении операции удаления в исходной по целевой директории из первой удаляются только файлы, совпадающие по названию с файлами во второй.

Функциональность

  • Возможность обновления и обработки исходной директории со слишком большим количеством файлов при умеренном потреблении ресурсов операционной системы
  • Возможность настройки фильтров при обновлении, обработке и удалении из исходной директории со слишком большим количеством файлов
  • Возможность удаления файлов из исходной директории со слишком большим количеством файлов - только выделенных или всех по фильтру
  • Создание Zip-архивов по списку директорий
  • Гибкая настройка имени Zip-файла при обработке исходной директории, создании Zip-архива целевой директории или архивов по списку директорий
  • Одновременная работа с разными возможностями основного функционала программы без блокирования пользовательского интерфейса
  • Интерактивное отображение состояния выполняемых операций, лога обработки исходной директории
  • Сохраняемые настройки пользовательского интерфейса, возможность сохранения выбранных директорий для повторного обращения
  • Возможность навигации по директориям и удаления файлов для целевой директории
  • Проверка корректности пути к выбранной директории
  • Просмотр файлов целевой директории в самой программе
  • Возможность определения свободного места на ресурсе исходной директории

О программе

WordsBrowser - программа для изучения слов и выражений английского языка с помощью популярных онлайн-словарей.

С помощью данной программы вы можете быстро сохранить слово, в том числе составное - например, фразовый глагол или выражение - и регулярно повторять его с целью изучения и запоминания.

Системные требования

Операционные системы: Windows XP, Windows 7.

Дополнительно: Microsoft .Net Framework 4.

Описание работы

Основное назначение программы WordsBrowser - помощь в изучении слов и выражений английского языка, используя сайты поддерживаемых онлайн-словарей.

Чтобы сохранить слово для последующего изучения, нужно перейти на сайт онлайн-переводчика и, получив перевод изучаемого слова, добавить его в Закладки.

В дальнейшем слово можно изучать либо через Повтор, слов либо с помощью Словаря самой программы.

Основные возможности программы подробно описаны ниже. Полный функционал программы представлен в разделе "Функциональность"

Начало работы с программой

Программа не требует установки, достаточно скачать Zip-архив по предлагаемой ссылке и развернуть в любой удобной директории. При первом запуске программы, после принятия условий Лицензионного соглашения Вам будет предложено добавить поддерживаемые программой сайты онлайн-словарей. Рекомендуется оставить все предложенные варианты и нажать Ok.

После открытия программы добавленные словари будут находиться в папке Сайты на левой боковой панели Закладок. Вы всегда можете добавить в эту папку ссылку на любой сайт, однако для поддерживаемых онлайн-словарей название новой закладки автоматически определяется по ссылке на страницу с переводом слова. Автоопределение реализовано в плагинах, каждый из которых соответствуюет своему онлайн-словарю. На данный момент используются следующие плагины:

  • Wiktionary - для использования с English-language Wiktionary.
  • Yandex - для использования с Яндекс.Словари.
  • LingvoOnline - для использования с Онлайн-словарь ABBYY Lingvo-Online.

При первом запуске программы загружаются все плагины. Если Вы предпочитаете пользоваться не всеми из предложенных онлайн-переводчиков, всегда можно выгрузить ненужные плагины из памяти через меню Инструменты > Настройки, вкладка Плагины.

Добавление слова в Закладки

Перейдите на сайт онлайн-переводчика и, получив перевод изучаемого слова, добавьте ссылку в Закладки одним из предлагаемых способов:

  • Нажав на кнопку на Панели навигации
  • Меню Закладки > Добавить слово
  • Сочетанием клавиш Ctrl+D

Если сайт, с которого получен перевод, принадлежит к поддерживаемым, в форме "Добавление в Закладки" поле Слово будет заполнено автоматически. Иначе поле будет пустым и придется заполнить его самостоятельно.

Закладки слов хранятся в папках, соответствующих датам. Поэтому в форме добавления слова можно выбрать дату, к которой относится сохраняемое слово. По умолчанию выбирается текущая дата, однако из выпадающего списка можно выбрать папку уже существующей даты. Если папка не существует, она будет создана. Используя контекстное меню, сохраненную закладку в дальнейшем можно:

  • Открыть
  • Открыть в новой вкладке
  • Переименовать
  • Удалить

Удалить можно также и всю папку, содержащую слово. Перед удалением программа выдаст предупреждение, т.к. после удаления восстановить папку и все входящие в нее закладки будет невозможно.

Повтор слов

Сохраненные ссылки на слова с переводом можно повторять. Пользовательский интерфейс по работе с Повтором слов находится в Панели повтора и в меню Закладки > Повтор слов.

Процесс повтора слов заключается в формировании состава повторяемых слов и последовательном переходе на ссылки через заданный интервал времени. Состав, сортировка и интервал повтора задаются в форме Настройки повтора слов в Панели повтора. Заданные сортировка и интервал сохраняются как пользовательские настройки.

Работа со Словарем

Повторять слова можно также используя Словарь на Панели поиска или в меню Закладки > Словарь.

Откройте форму, найдите нужные слова, отметьте их и нажмите Показать выделенное. Ссылки на слова будут отображены на новых вкладках.

Настройки

В форме Настройки меню Инструменты на первой вкладке можно сохранить Домашнюю страницу и отключить Заставку. На второй вкладке формы можно настроить Плагины.

История

Программа хранит историю посещенных сайтов. Журнал располагается на второй вкладке Боковой панели и дает возможность представления истории По дате, По сайту и По порядку посещения.

Прочее

Вы можете осуществлять навигацию по папкам с помощью Календаря. Для этого откройте меню Закладки > Календарь. Даты, для которых существуют папки закладок, будут выделены жирным шрифтом.

В Панели поиска можно пользоваться поисковыми системами Google, Yandex или Bing.

В меню Инструменты можно вызвать Калькулятор.
