О программе

PlaneFileDistributor - легковесный файловый менеджер для автоматизации стандартных операций с файлами.

Программа позволяет скомпоновать типичные команды - копировать, переместить и удалить - для файлов в исходной папке и выполнить их за один клик. Исходной папкой может быть, например, папка, в которую вы скачиваете файлы из Интернета, или любая другая папка, из которой Вы хотите распределить файлы.

Программа может быть особенно полезной, когда файлы необходимо распределить в более чем одну папку.

Системные требования

Операционные системы: Windows 7 -10.

Дополнительно: Microsoft .Net Framework 4.5.1

Описание работы

Задайте исходную папку, добавьте папки, в которые Вы хотите распределять файлы из исходной, и программа готова к работе. Компонуйте стандартные команды - копировать, переместить и удалить и распределяйте файлы за 1 клик.

Основные возможности программы подробно описаны ниже. Полный функционал программы представлен в разделе "Функциональность"

Начало работы с программой

Программа не требует установки, достаточно скачать Zip-архив по предлагаемой ссылке и развернуть в любой удобной директории. При первом запуске программы необходимо будет принять условия Лицензионного соглашения.

Для начала нужно выбрать к путь к исходной папке - это делается на левой панели. При первом запуске программы в качестве исходной устанавливается пользовательская папка Мои документы. Изменение папки проводится кнопкой "..." либо вставкой пути к файлу через буфер обмена. В последнем случае после вставки нажмите клавишу "Enter". Открыть файл можно дважды кликнув по строке с информацией о файле.

Файлы в исходной папке можно фильтровать по имени, допускается совпадение с любой частью файла.

При изменении состава файлов в исходной папке автоматического обновления не происходит. Для обновления вида нажмите кнопку (или комбинацию клавиш Ctrl+R).

Задание файловых команд

Создайте файловые команды исходя из того, что Вы хотите сделать с файлами в исходной папке, в меню Новая команда.

Команды Копировать и Переместить имеют 3 опции в зависимости от действия, которое осуществляется при совпадении по названию файлов в исходной папке и папке назначения:

  • ошибка - при совпадении ошибка, файл из исходной папки не копируется(не перемещается)
  • пропустить - при совпадении файл из исходной папки пропускается и не копируется(не перемещается)
  • заменить - при совпадении файл в папке назначения будет заменен файлом из исходной папки

Опции команд Копировать и Переместить, относящиеся к совпадению файлов, можно изменить - кнопкой Изменить или двойным кликом на команде - через карточку команды.

Команда Удалить имеет 2 опции в зависимости от необходимости подтверждения перед выполнением удаления:

  • с подтверждением - при выполнении команды нужно будет подтвердить удаление файлов
  • без подтверждения - файлы будут удалены без дополнительного подтверждения

Опция подтверждения команды Удалить меняется на нижней панели после установки фокуса на строку с командой.

Внимание! В отличие от большинства файловых менеджеров файлы будут удалены не в Корзину, а безвозвратно!

Команда Копировать и удалить имеет те же опции, что и команды Копировать и Переместить(за исключением опции “пропустить” ). После завершения копирования в выбранные папки назначения данная команда, в случае отсутствия ошибок при копировании, удаляет файлы в исходной папке.

Для текущих опций команд назначены следующие быстрые клавиши:

  • F5 - Копировать
  • F6 - Переместить
  • F8 - Удалить
  • F9 - Копировать и удалить

После смены опции быстрая клавиша переназначается на новую опцию.

Выбор файлов для команд, затрагивающих существование файла в исходной папке, таких как перемещение и удаление, приводит к исключению возможности выбора этих файлов для последующих команд в списке. Строки с такими файлами становятся затененными, а выделение флажками - недоступным.

Удалить команды можно кнопкой Удалить, либо выделив необходимые команды и нажав клавишу Delete.

Избранное

Исходные папки можно добавлять в Избранное. Для добавления текущей папки нажмите кнопку.

Для редактирования Избранного нажмите кнопку.

Папки назначения

После добавления хотя бы одной команды Копировать или Переместить можно задавать папки назначения.

Для добавления папки воспользуйтесь кнопкой панели инструментов списка папок назначения. В появившемся окне карточки задайте короткое имя и путь к папке. Короткое имя позволяет назначить папке своего рода псевдоним, который можно использовать в дальнейшем для быстрой идентификации папки.

Для изменения папки воспользуйтесь кнопкой панели инструментов, либо дважды кликните на строку с данными папки.

Для удаления папок воспользуйтесь кнопкой панели инструментов, либо выделите необходимые папки и нажмите клавишу Delete.

Вы можете редактировать состав папок назначения как работая с текущей командой, так и через отдельный справочник по кнопке в панели инструментов.

Для просмотра содержимого папки в Проводнике Windows нажмите кнопку .

Выполнение файловых команд

Для каждой команды отметьте флажками файлы на левой панели(вы можете отметить несколько файлов за раз, предварительно выделив их, а также воспользоваться быстрыми клавишами Enter для отметки флажками и Shift+Enter для снятия), для команд Копировать и Переместить - задайте папки назначения (также с помощью выделения флажками на панели под списком команд) и нажмите кнопку Применить(Alt+R). Перед выполнением, если будут найдены команды с незаданными файлами или папками, программа выдаст предупреждающее сообщение о том, что эти команды выполнены не будут. Вернуться к списку команд и задать необходимые параметры можно выбрав Отмена в диалоговом окне подтверждения. Применение команд происходит в асинхронном режиме, в это время Вы можете работать с Избранным или свернуть программу в панель задач.

Результат выполнения текущей команды отображается в строке состояния в кратком виде - операция, количество файлов в ней и короткое имя папки назначения, например - "скопировано 1 в Backup". При наличии ошибок выдается сообщение с указанием операции, файла, папки назначения и описания ошибки.

Результаты операций - как положительные, так и ошибки - логируются для каждого файла. Посмотреть лог-файл можно через меню Лог.

По окончании выполнения всех команд их список очищается, а исходная папка обновляется. Окно программы активируется и на нижней панели отображается лог только что примененных команд.

Групповое переименование файлов*

Выделите флажками файлы, которые нужно переименовать, и нажмите Shift+F6 либо кнопку на панели инструментов главной формы. Вы можете осуществлять 2 разных вида переименования - добавить символы в начало либо конец имени файла или заменить символы, входящие в имя файла. Эти 2 вида переименования уже представлены на форме, Вы можете добавить новое переименование, нажав на кнопку со знаком "+", либо удалить существующее переименование. Если ключевые поля переименования (хотя бы одно для добавления и начальное для замены) не заполнены, оно не используется. Все произошедшие переименования логируются.

Работа с тэгами*

Для редактирования тэгов создайте любую команду(кроме Удалить) или откройте справочник папок назначения. В панели инструментов нажмите кнопку . Вы можете привязывать тэги к любой папке назначения. Для этого откройте карточку папки и выберите флажком необходимые тэги. Вы можете фильтровать отображаемый список папок назначения любой команды (кроме Удалить), выбирая тэги в нижней части панели. В справочнике папок назначения тэги, привязанные к папке, показываются через запятую на нижней части формы.

Миниатюры файлов

Наведите мышь на иконку файла. Примерно через секунду появится всплывающая подсказка с названием, размером и миниатюрой файла или иконкой, если миниатюру получить не удалось. Просмотр миниатюр реализован для большинства известных типов файлов.

Представления файлов в исходной папке

Помимо стандартного представления файлов в виде иконок, которое отображается при загрузке и соответствует кнопке на панели инструментов, нажав на кнопку можно выбрать представление в виде миниатюр. Миниатюра формируется так же, как описано в предыдущем пункте. Для ряда типов файлов, например с расширением .htm(l), в случае их достаточно большого количества, отображение может занимать большое время. Для отмены процесса отображения миниатюр нажмите кнопку на панели инструментов.

Особые случаи

Если копирование файла завершилось с ошибкой либо было пропущено, и за этой командой следует команда удаления или перемещения, исчезновение файла в исходной папке как минимум нежелательно. Программа учитывает такие случаи и не допускает того, чтобы проблемные файлы исчезли из исходной папки. При этом для каждого проблемного файла при выполнении последующих команд удаления или перемещения в лог-файл будет добавлена запись вида : "Пропущено при удалении, выполнение предыдущей команды произошло с ошибкой или удаление недопустимо", с указанием имени файла, исходной папки, и, в случае перемещения, папки назначения.

Если вы хотите просто удалить файлы в исходной папке, отметьте их флажками и нажмите клавиши Shift+Del.

Функциональность

  • Информация о файлах в исходной папки с возможностью открытия по двойному клику.
  • Фильтрация файлов в исходной папке.
  • Возможность задать папки для распределения файлов.
  • Возможность компоновки основных файловых команд и выполнения их за 1 клик.
  • Фильтрация папок назначения по тэгам.*
  • Возможность просмотра миниатюр файлов.
  • Групповое переименование файлов*.
  • Возможность представления файлов в исходной папке в виде миниатюр.
  • Контроль ошибок при выполнении команд.
  • Ведение общего лога и отображение лога примененных команд.
